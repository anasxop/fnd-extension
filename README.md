# FND Browser Extension

A browser extension that verifies news articles using AI. Built as part of a student project for Fake News Detection (FND).

---

## Features

- One-click news verification
- AI-powered analysis
- Works on any website
- Clean and minimal UI
- Connected to FND backend
- Supports Chrome & Opera GX

---

## How It Works

1. Open any news article
2. Click the FND extension icon
3. The extension analyzes the content
4. You get a Real/Fake verdict instantly

---

## Installation (Manual)

Since this is a student project, the extension is installed manually:

### Chrome / Opera GX

1. Download this repository as ZIP
2. Extract the ZIP file
3. Open browser and go to:
   - Chrome: `chrome://extensions`
   - Opera GX: `opera://extensions`
4. Enable **Developer Mode**
5. Click **Load unpacked**
6. Select the extracted folder
7. Done 🎉
